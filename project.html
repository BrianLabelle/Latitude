Project Details
  • Landing Page contains: explanation of project & links to each visualizations page.

  • 4 visualization pages: each containging title, heading, plots and a descriptive paragraph

  • Comparison page : contains all visualizations as a bootstrap grid.

  • Data page : contains a responsive bootstrap table that was populated from Python Pandas

The project has:

  • Has the name of the site on the left of the nav which allows users to return to the landing page from any page.

  • Contains a dropdown on the right of the navbar named "Plots" which provides links to each individual visualization page.

  • Provides two more links on the right: "Comparisons" which links to the comparisons page, and "Data" which links to the data page.

  • Is responsive (using media queries). ( Inside of the custom.min.css ) file. @media (min-width: 768px), etc...

  • The website must be deployed to GitHub pages. brianlabelle.github.io/Latitude
